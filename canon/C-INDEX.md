## How to Use This Index

This document is the compass of the GAIA canon. Every active document in the `canon/` directory is listed here, grouped by thematic series, with a brief description of each document's core contribution. Documents marked `[DEPRECATED]` or `[ARCHIVED]` are preserved for historical continuity but superseded by the active version listed.

Cross-references noted as `→ see also` point to documents in adjacent series that speak to the same territory from a different angle.
