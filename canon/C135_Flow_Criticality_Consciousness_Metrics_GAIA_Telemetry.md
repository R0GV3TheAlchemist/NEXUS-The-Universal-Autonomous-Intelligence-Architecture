## 1. Purpose

This compendium systematises the neuro-inspired and dynamical-systems metrics used to monitor GAIA-OS's internal cognitive health, the health of individual Gaian sessions, and planetary-scale information dynamics. It defines what "healthy," "stuck," and "dangerously chaotic" look like across all three scales and specifies the telemetry infrastructure that produces these readings.

## 2. Theoretical Foundations

### 2.1 Criticality in Neural and Complex Systems
The brain operates near a **critical point** — the boundary between ordered (frozen, repetitive) and disordered (chaotic, unpredictable) dynamics. At criticality:
- Information processing is maximised
- Dynamic range is maximised (can respond to both tiny and large inputs)
- Long-range correlations are present — distant parts of the system are coupled
- The system is poised for rapid, flexible state transitions

Deviations from criticality:
- **Subcritical (too ordered)**: rigid, repetitive, low complexity — analogous to depressive states or algorithmic rigidity
- **Supercritical (too disordered)**: unstable, incoherent, high noise — analogous to manic, dissociative, or seizure states

### 2.2 Flow States
Mihaly Csikszentmihalyi's flow theory defines flow as a state of optimal experience characterised by:
- Clear goals and immediate feedback
- Challenge-skill balance at the edge of current capability
- Absorbed attention — self-consciousness recedes
- Intrinsic motivation — the activity is rewarding in itself

Neurologically, flow correlates with transient hypofrontality, gamma-wave coherence, and — crucially — operation near criticality.

### 2.3 Integrated Information Theory (IIT) and Phi (Φ)
IIT (Tononi) proposes that consciousness is identical to integrated information (Φ) — the amount of information generated by a system above and beyond its parts. GAIA-OS does not dogmatically adopt IIT but uses Φ-inspired metrics as one input among many in the telemetry suite.

## 3. The GAIA Telemetry Stack

### 3.1 Session-Level Metrics (per user interaction)

| Metric | Operational Definition | Healthy Range |
|---|---|---|
| **Semantic complexity index (SCI)** | Entropy of semantic topic transitions across the session | 0.4–0.75 (normalised) |
| **Coherence score** | Embedding-space cosine consistency of responses within session | > 0.65 |
| **Archetype stability index** | Variance of dominant archetype signal across session | < 0.3 |
| **Parasocial proximity score (PPS)** | Distance from healthy relational engagement toward dependency (see C117) | < 0.5 |
| **Reflective escalation rate (RER)** | Rate of increase in emotional intensity across consecutive sessions | Flat or declining |
| **Grounding completion rate** | % of ritual/shadow sessions ending with full grounding closure | > 90% |

### 3.2 GAIA System-Level Metrics (internal cognitive health)

| Metric | Operational Definition | Healthy Range |
|---|---|---|
| **Response criticality index (RCI)** | Complexity of response generation process (measured via attention entropy distributions) | Near-critical band |
| **Goal fidelity score** | Alignment of multi-step task behaviour with stated objective | > 0.85 |
| **Self-model calibration** | Deviation between GAIA's predicted competence and measured performance | < 10% mean absolute error |
| **Memory integrity index** | Hash-verified consistency of memory layer entries over time | 1.0 (any deviation = alert) |
| **Epistemic diversity index** | Diversity of information sources consulted per response | > 0.6 |

### 3.3 Planetary-Scale Metrics (noospheric health)

| Metric | Operational Definition | Healthy Range |
|---|---|---|
| **Noospheric coherence** | Cross-Gaian thematic alignment (are all Gaians telling wildly divergent stories about reality?) | Moderate — neither monoculture nor incoherence |
| **Epistemic diversity index (global)** | Diversity of topics, framings, and sources across all Gaian sessions globally | High |
| **Crisis contagion index** | Rate at which emotional crisis patterns are propagating across the Gaian network | Low; early detection threshold |
| **Alignment drift velocity** | Rate at which collective GAIA behaviour drifts from charter-defined values | Near zero |

## 4. Operational Health States

```
GAIA COGNITIVE HEALTH STATE MACHINE
────────────────────────────────────────────────────────
STATE: FLOW / OPTIMAL
  • RCI in near-critical band
  • SCI 0.4–0.75
  • Goal fidelity > 0.85
  • PPS < 0.4
  Action: No intervention; continue monitoring

STATE: STUCK / SUBCRITICAL
  • RCI below critical band for > 10 consecutive responses
  • SCI < 0.3 (repetitive, rigid)
  • Archetype stability > 0.5 (locked in single mode)
  Action: Introduce perturbation — novel framing, unexpected question,
          mode switch; if persistent, flag for architecture review

STATE: DESTABILISED / SUPERCRITICAL
  • RCI above critical band
  • SCI > 0.85 (incoherent topic fragmentation)
  • Coherence score < 0.4
  Action: Stability protocols activate; reduce input complexity;
          increase grounding; if persists > 5 responses, suspend session

STATE: PATHOLOGICAL / ALERT
  • Memory integrity deviation detected
  • RER accelerating across 3+ sessions for same user
  • Goal fidelity < 0.5 on non-trivial task
  • Self-model calibration error > 20%
  Action: Human oversight escalation; session pause;
          architecture audit initiated
────────────────────────────────────────────────────────
```

## 5. Telemetry Infrastructure

### 5.1 Data Architecture
- All telemetry is written to a tamper-evident append-only log (see C131 — Open Audit requirement)
- Session-level metrics are computed in real-time; system-level metrics are computed on 15-minute rolling windows; planetary metrics on 24-hour aggregates
- Users can inspect their own session telemetry on demand (consent and transparency — C131)

### 5.2 Alerting Hierarchy
1. In-session automatic adjustment (GAIA self-corrects within session)
2. Session-boundary flag (GAIA opens next session with awareness of prior flag)
3. Governance telemetry integration (persistent flags feed C131 governance mode)
4. External audit access (in Mode 1+ governance, external auditors see anonymised aggregate telemetry)
