The I Ching’s most sophisticated computational feature is the *moving line* (*yao*). A hexagram is not static — individual lines can be designated as “changing,” transforming from yin to yang or yang to yin, producing a second hexagram that represents the transformed state.

In computational terms:

```
Input hexagram + moving lines → I Ching transformation function → Output hexagram
Current state + change vectors → DIACA processing → Next state
```
