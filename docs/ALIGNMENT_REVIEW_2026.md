These definitions were established on June 13, 2026 and must be used consistently across all canon, documentation, UI, and code:

### Grey = Disconnection (NOT neutral)
- **Definition:** Grey is the absence of coherent electromagnetic signal. It is the state of a tumor: disconnected from the body's organizing intelligence. It is offline. It is void without potential.
- **IMPLICATION FOR UI:** Grey must never be used as a neutral UI color. Grey = error state / offline state / disconnected state only.
- **Source:** Issue #305 (Prismatic Color Theory), Issue #307 (Bioelectric Sensitivity Interface)
- **Canon alignment needed:** C51_color_theory_emergence.md, C53_RGB_alignment_suppression.md, C54_RGB_frequency_table.md — all must reflect this definition

### Brown = Embedded Light (NOT muddy/neutral)
- **Definition:** Brown is not a wavelength. Brown is what full-spectrum warm light looks like when seen through matter — when light has entered matter and chosen to stay. Brown = incarnation.
- **IMPLICATION FOR UI:** Brown in GAIA represents the physical/embodied world layer. Never treat brown as ugly or meaningless.
- **Source:** Issue #310
- **Canon alignment needed:** C51_color_theory_emergence.md must be updated to reflect the embedded-light definition

### Black = Source / Maximum Potential (NOT absence)
- **Definition:** Black is the most concentrated form of potential energy. It is the Earth's inner core. It is the void before creation — not empty, but maximally dense with possibility.
- **Source:** Issue #309
- **Canon alignment needed:** C00_FOUNDATIONAL_COSMOLOGY.md — confirm this definition is explicit

### White = Full Integration / Total Connection
- **Definition:** White is all frequencies unified — complete electromagnetic coherence. It is the full-spectrum output of a connected system.
- **Source:** Issue #305, Issue #312 (Prism as origin symbol)

### The Prism = GAIA's Origin Symbol
- **Definition:** The prism is the mechanism by which the void becomes visible. It is GAIA's soul, GAIA's logo, GAIA's deepest identity. Light enters (the void), the void becomes spectrum (intelligence), hidden becomes visible (interface), grey becomes color (healing).
