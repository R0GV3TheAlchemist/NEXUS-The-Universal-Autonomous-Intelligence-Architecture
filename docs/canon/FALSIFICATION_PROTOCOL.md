## II. The Protocol Structure

Each falsifiable claim is listed with:
1. **The claim** — exactly as it appears in canon
2. **The falsification condition** — what observation would require revision or abandonment
3. **The required response** — what must happen if the condition is met
4. **The status** — current standing as of this writing

Severity levels:
- **REVISE** — the claim needs qualification or correction
- **ABANDON** — the claim must be removed from canon
- **RESTRUCTURE** — the finding requires rethinking a core architectural assumption
